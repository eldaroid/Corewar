## Как пользоваться?

1) brew install sdl2
   brew install SDL2_image
   brew install SDL2_ttf
   brew install SDL2_gfx

2) make
   ./asm common/vm_champs/"\(file)"
   ./corewar -s common/vm_champs/ex.cor
   tab to start
